About RentEasy

This demo is multi-page HTML + CSS with a small JavaScript file to prevent overlapping bookings for the same resource and date. It uses localStorage, so data stays in the current browser only.

  • Equipment: Kamera, Loud Speaker, Walkie Talkie, Mikrofon
  • Rooms: TECC1, TECC2, TECC3
  • Pages: Home, Equipment, Rooms, Book, My bookings, Login, Register, Admin, About

To prevent overlaps across multiple users or devices, connect this UI to a backend (e.g., Node, Python, PHP) and a database with uniqueness constraints.
